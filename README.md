## How to use

### Browser

```js
<script src="lib/WMDeflate.js"></script>
<script>
console.log( WMDeflate() );
</script>
```

### WebWorkers

```js
importScripts("lib/WMDeflate.js");

console.log( WMDeflate() );
```

### Node.js

```js
var WMDeflate = require("lib/WMDeflate.js");

console.log( WMDeflate() );
```
